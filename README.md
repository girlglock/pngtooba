# PNGTooba

An OBS PNGTuber plugin with vowel support and simple effects

<img src="https://kappa.lol/JpctyN.avif">

**Features:**
- idle/speaking images
- vowel mouth shapes
- shake/bounce/hop effects

**Installation:**
- extract the zip into your obs install (eg `C:\Program Files\obs-studio`)
- in OBS, add a new source and choose **PNGTooba**
